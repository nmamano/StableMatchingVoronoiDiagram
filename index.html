Usage

Click on a site (the little black circle) to select it. A selected site can be dragged or deleted by pressing the back or DEL keys. Create new sites or delete existing sites with double click.

The square has a total area of 100 appetite units. If the number of sites times the appetite is larger than 100, not all sites will be satisfied.

About

This tool allows to visualize stable-matching Voronoi diagrams. It is made to complement the paper Stable-matching Voronoi Diagram: Combinatorial Complexity and Algorithms. However, this website does not use the algorithm from the paper. Instead, it uses a simpler algorithm for a pixelated plane rather than a continuous plane. Developed by Nil Mamano.
